# OpenGraph Default Image

## Required File
Please create and place a file named `og-default.jpg` in this directory (`/public/`).

## Image Specifications
- **Dimensions:** 1200 x 630 pixels (optimal for OpenGraph)
- **Format:** JPG or PNG
- **File Size:** Under 1MB recommended
- **Content:** Should include:
  - FansClubOnly branding/logo
  - Tagline: "Discover Top OnlyFans Creators"
  - Clean, professional design
  - High contrast for visibility

## Usage
This image will be used as the default social media preview image when:
- Sharing the homepage
- Sharing pages without specific images
- Fallback for creator profiles without profile images

## Current Path
The image should be located at: `/public/og-default.jpg`

Which will be accessible at: `https://fansclubonly.com/og-default.jpg`
